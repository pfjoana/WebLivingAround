# Living Around

Website para aluguer de autocaravanas, desenvolvido como projeto do módulo de Programação Web do curso Software Developer.

## Descrição

Living Around é uma plataforma web que permite aos utilizadores explorar e alugar autocaravanas para as suas viagens. O site oferece uma experiência visual atrativa e intuitiva, com informações detalhadas sobre cada veículo, localização dos pontos de recolha e um formulário de contacto.

## Funcionalidades

- **Homepage**: Página inicial com apresentação do serviço e navegação para as diferentes secções
- **Acomodações**: Catálogo de autocaravanas disponíveis com informações detalhadas
- **Locais**: Mapa interativo com localização dos pontos de recolha nas diferentes cidades
- **Contactos**: Formulário de contacto com feedback de envio através de modal

## Tecnologias Utilizadas
- **HTML**: Estrutura das páginas
- **CSS**: Estilização e layout
- **Bootstrap**: Framework CSS  
- **Google Maps**: Integração de mapa para visualização de localizações
- **Google Fonts**: Tipografias Jura e Josefin Slab
